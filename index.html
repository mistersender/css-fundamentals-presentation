Selectors

How CSS knows what elements to apply rules to

element Selectors

Selects all elements

HTML: 



  I'll be green
CSS: 

h1 {
  color: green;
}

.class Selectors

Selects all elements with a particular class

HTML: 



  I'll be green
CSS: 

.my-class-selector {
  color: green;
}

#id Selectors

Selects all elements with a particular ID

HTML: 



  I'll be green
CSS: 

#my-id-selector {
  color: green;
}

Enter: The Cascade

We have to have a way to tell which styles take prescedence

Cascade: Order

"The closer to the element, the more I care."

Physically lower styles take prescedence


h1 {
  color: green;
}
div {
  color: blue;
}
h1 {
  color: orange; /* h1 will be orange */
}

Styles on elements take prescedence


h1 {
  color: green;
}
          


I'll be purple.

USE SPARINGLY!
